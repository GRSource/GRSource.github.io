# 一般来说，I帧的压缩率是7，P是20，B是50

# AVFrame

AVFrame结构体 来描述解码后或编码前的原始数据，对于视频来说，AVFrame就是视频 的一帧图像

# AVPacket

FFmpeg中使用 AVPacket结构体来描述解码前或编码后的压缩数据

# YUV

Y表示亮度，通过RGB信号的特定部分叠加计算得来。 UV定义了颜色的两个方面色调和饱和度，Cr反映了红色部分与亮度之间的差异，Cb反映了蓝色部分和亮度之间的差异

# Rec.601和BT.709中，Y范围为16-235，UV范围为16-240
