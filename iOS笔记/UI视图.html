# 事件传递&视图响应

事件传递： 点击屏幕 --> UIApplication --> UIWindow --> hitTest:withEvent: --> 内部通过pointInside:withEvent:判断是否在响应范围之内 --> 倒序遍历子视图(最后添加的视图优先遍历)-->每个UIView中都会调用hitTest方法-->如果hitTest返回的某个view不为空，那么久找到最终的响应视图，事件传递就结束。 响应流程： UIlabel ->UIView -> UIView ->UIViewController ->UIWindow ->UIApplication ->UIApplicationDelegate -->touchesBegan:withEvent: touchesMoved:withEvent: touchesEnded:withEvent: 如果传递到UIApplicationDelegate都没处理，就丢掉此次响应
